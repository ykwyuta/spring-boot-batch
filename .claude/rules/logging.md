# ログ記録ルール

## スキル・サブエージェントの作成・改修ログ

- 場所: `.claude/logs/skills-YYYYMMDD-HHMMSS.log`
- 対象: スキル（`.claude/skills/`）またはサブエージェント（`.claude/agents/`）の新規作成・改修・削除（廃止）を行った都度、1 ファイルを作成する。
- 記載内容の例:
  - 日時
  - 対象（スキル名 / エージェント名、ファイルパス）
  - 変更種別（新規作成 / 改修 / 廃止）
  - 変更内容の概要
  - 変更理由

```
日時: 2026-06-27 12:00:00
対象: .claude/skills/example-skill/SKILL.md
種別: 新規作成
概要: XXXを行うスキルを追加
理由: ユーザーからの依頼によりXXXの作業を自動化するため
```

## ユーザー指摘に基づく作業履歴ログ

- 場所: `logs/claude-YYYYMMDD-HHMMSS.log`
- 対象: ユーザーからの指摘・依頼を受けて実施した作業全般。
- 記載内容の例:
  - 日時
  - ユーザーからの指摘・依頼内容の要約
  - 実施した作業内容
  - 変更したファイル一覧

```
日時: 2026-06-27 12:00:00
指摘内容: CLAUDE.mdとルールファイルの整備
実施内容: CLAUDE.mdおよび.claude/rules/以下のルールファイルを作成
変更ファイル: CLAUDE.md, .claude/rules/*.md
```
